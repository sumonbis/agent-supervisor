## The job has changed

AI coding agents now plan, edit files, run commands, and report "done" on their own.
Your role has shifted from **writing** code to **supervising** an agent that writes it.

Open the **Supervision Trainer** to step into that seat. You'll watch real agent
behavior and practice the six skills of a good supervisor: Direct, Approve, Verify,
Critique, Own, and Orchestrate.
